Throubleshooting

If you see this message, it probably means you've encountered an error.

What happened?

Your web browser blocked your request to RA Library application.
The most possible reason is that you didn't install RA certificate. It often happens when you're using personal mobile devices and it's the first time you visit RA hosted HTTPS web sites.
Or you've lost network connection.

How to resolve?

Visit this site via your web browser, and accept the certificate.
You only need to do this once. And make sure to check the certificate details before accept it!

Why do I need do this?

Because we serve this application on RA server through HTTPS.
